## Technical
For development purposes use `js/main.js` instead of `js/main.min.js`.

In order to optimise, you must have _node.js_ installed. Then do the following:
- `$ cd js/tools`,
- `$ node r.js -o build.js`,
- `$ cd ../../` to return to the main directory
